
# 📱 QR Code Component

A simple and responsive QR Code card built using **HTML** and **CSS**.  
This project is based on a Frontend Mentor challenge and is perfect for beginners to practice layout and responsiveness.

---

## 🚀 Live Demo
👉 https://khushboorawat109-bot.github.io/qr-code-component/

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox)
- Responsive Design (Media Queries)

---

## ✨ Features

- Clean and minimal UI
- Fully responsive (mobile + desktop)
- Centered layout using Flexbox
- Modern card design with shadow and rounded corners

---
